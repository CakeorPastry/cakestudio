How To Use URL Params?


https://website.com?firstparam=value&second=value...
Cyan = First param with it's value
Green = Second param with it's value
... = You can add more params. They can go in any order. The rule is that the first param starts with ?
and the remaining params (i.e. second, third, fourth and so on...) will start with &
